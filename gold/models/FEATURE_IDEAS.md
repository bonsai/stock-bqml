# Feature Ideas (未実装)

収集した特徴量アイデア。実装時に `silver/features_daily.sql` に追加。

## カレンダー効果

| 特徴量 | アイデア | データ元 |
|---|---|---|
| 曜日ダミー | 火曜・水曜は上がる傾向。`dow_tue`, `dow_wed` BOOL | `EXTRACT(DAYOFWEEK FROM date)` |
| 月ダミー | `month_jan` ... `month_dec`。決算期(3月/9月)の季節性 | `EXTRACT(MONTH FROM date)` |
| 週番号 | 月初/月末、第n週目のパターン | `EXTRACT(WEEK FROM date)`, 月初からの経過日数 |
| 四半期末 | 機関投資家のポートフォリオ調整売り | `EXTRACT(QUARTER FROM date)` かつ月最終週 |
| 連休前売り | バカンス前にポジション圧縮。データドリブン: 過去の連休前リターン移動平均 | 祝日カレンダー + rolling window |
| SQ週 | 特別清算指数(第2金曜)はオプション到期で異常ボラ | 第2金曜計算 + 前後5営業日 |

## 決算関連

| 特徴量 | アイデア | データ元 |
|---|---|---|
| 決算前売り | 決算日前は不確実性回避で売られる。決算日からN日前 | 決算日カレンダー (外部データ) |
| 決算後ギャップ | 決算発表後のギャップ幅 | 同上 + 価格データ |
| 配当落ち調整 | 配当権利落ち日の価格調整 | 配当カレンダー |

## 市場全体（投資強気/弱気）

| 特徴量 | アイデア | データ元 |
|---|---|---|
| 騰落レシオ | 値上がり数 / 値下がり数。市場全体の過熱感 | 東証データ or 個別銘柄集合から近似 |
| 日経平均との相関 | 銘柄 vs 日経平均の rolling 相関 | インデックスデータ |
| セクター強弱 | 所属セクターの平均リターンと銘柄の乖離 | セクター分類 |
| 市場ボラ | VIX 相当の日本版（日経VI） | 外部API |

## 実装メモ

- カレンダーベースの特徴量は `reference/jp_calendar.sql` で事実テーブルを作成してから結合
- データドリブン版は `PREV(n)_holiday_return_avg` のように rolling window で統計量を計算
- 「決算前売り」は決算日テーブルが必要 → 外部データ or ユーザー入力
- 曜日・月・週は `EXTRACT()` のみで追加可能、データ参照不要
